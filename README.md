# All in One for Jetson

All in One Jetson script to install latest version of TensorFlow, PyTorch, ROS, ... 

Machine Learning Frameworks for Jetson and JetPack

Update July 2021

Based on JetPack 4.5.1

Tested on Jetson Nano

   ``` 
   sudo sh allinone.sh
   ```
